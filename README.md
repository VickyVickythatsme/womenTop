## Installation & Setup Instruction
Clone down this repository. You will need `node` and `npm` installed globally on your machine. 
### a. Installation 
`npm install`
### b. Run Project
`npm run`
### c. Build production
`npm run build`
### d. To visite App
`localhost:3010` <br>
or click index.html bundled by webpack <br>
`./dist/index.html`
